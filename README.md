⏳ Life in Weeks – Interactive Life Visualizer
An interactive web application that visualizes a person's life in weeks based on their birth date. Inspired by the concept of "Life in Weeks", this project helps users understand the passage of time through an intuitive grid representation and meaningful life statistics.
